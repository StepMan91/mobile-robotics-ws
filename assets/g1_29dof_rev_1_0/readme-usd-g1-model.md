this is the readme for the g1_29dof_rev_1_0 model

there are two files:
- g1_29dof_rev_1_0.usd
- g1_29dof_rev_1_0.usdz

the .usdz file is the compressed version of the .usd file   

the model is based on the g1_29dof_rev_1_0 model from the unitree repository

the model is a USDZ file that can be opened with any USD viewer

the model is a USDZ file that can be opened with any USD viewer
